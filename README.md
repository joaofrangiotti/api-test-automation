# Automacao com Cypress

Repositorio GitHub: api-test-automation

Projeto simples para automatizar testes de API com Cypress.

## O que testa

- GET para listar dados
- POST para criar dados
- status code
- resposta JSON

## Como rodar

1. Instale as dependencias com npm install.
2. Rode o Cypress com npm run cy:open.
3. Para executar no terminal, use npm run cy:run.

## Simulacao de resultado

| Cenario | Resultado esperado |
|---|---|
| GET listar dados | 200 |
| POST criar dados | 201 |
